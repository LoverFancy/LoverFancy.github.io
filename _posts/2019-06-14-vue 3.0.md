## vue 3.0展望

1. 重写虚拟dom
2. 静态树提升
3. 使用Proxy观察者机制取代Object.DefineProperty
4. 体积更小，压缩后10k
5. 可维护性，很多包解耦
6. 支持TS
7. 实验性支持Time Slicing和hooks支持
8. 不用 Class API 改用 Function-based API
